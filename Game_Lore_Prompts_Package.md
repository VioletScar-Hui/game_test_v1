# 🧛‍♂️ 《暗夜猎手：虚空幸存者》网页端游戏剧情设定提示词包

> **设计参考**：Vampire Survivors（吸血鬼幸存者）  
> **适用场景**：轻量级网页端 Roguelike 生存游戏  
> **文档版本**：v1.0  
> **风格定位**：哥特荒诞 × 家族复仇 × 借口剧情（Excuse Plot）

---

## 一、核心叙事架构（Core Narrative Framework）

### 1.1 世界观一句话
```
20XX年，被诅咒的[虚构地区名]，邪恶的[反派名]用[荒诞的邪恶力量]蹂躏了这片土地。
[主角家族]挺身而出，誓要终结其统治——如果他们能找到[反派名]的话。
```

### 1.2 叙事三幕结构（适配30分钟单局）

| 阶段 | 时间轴 | 叙事功能 | 玩家体验 |
|------|--------|----------|----------|
| **第一幕：觉醒** | 0-5分钟 | 交代基础冲突，角色入场，暗示荒诞性 | 弱叙事，强氛围，环境叙事 |
| **第二幕：迷失** | 5-20分钟 | 强化"寻找但找不到"的核心笑点，敌人越来越强，线索越来越离谱 | 通过道具描述、敌人命名传递剧情 |
| **第三幕：真相/崩溃** | 20-30分钟 | 揭示终极荒诞真相（或根本不揭示），死神/清算者登场 | 高潮叙事释放，死亡即叙事终点 |

### 1.3 核心叙事钩子（The Hook）
- **表面目标**：猎杀终极反派，拯救世界
- **实际目标**：在无尽的敌人潮中生存30分钟
- **隐藏笑点**：终极反派可能不存在，或根本不想被找到，或找到后发现是误会
- **元叙事**：游戏本身就在嘲讽"拯救世界"这一游戏 cliché

## 二、世界观设定（World Building）

### 2.1 基础设定模板
```markdown
**时间**：20XX年，近未来/架空现代
**地点**：[虚构地名，建议用双关或恶搞命名]，欧洲乡村/东方古镇/美式废土混搭
**核心灾难**：[反派名]使用了[荒诞的黑暗魔法/科技/诅咒]，导致：
- 大地被[奇怪的怪物]覆盖
- 食物变成了[搞笑的东西，如：会跑的烤鸡、发光的泡面]
- 时间每30分钟重置一次（解释Roguelike机制）
**文化基调**：哥特式建筑 + 现代网络梗 + 地方方言/口音文字化
```

### 2.2 反派设定：Bisconte Draculó 式模板
```markdown
**正式名称**：Count [姓氏]ó（如：Bisconte Draculó → 改为：Gourmando Feastó）
**身份**：末代贵族 / 堕落科学家 / 被解雇的厨师 / 过气网红
**罪行**：
- 用邪恶魔法制造了饥荒（但地上全是食物）
- 召唤了怪物大军（但怪物都很可爱/搞笑）
- 躲在某处（但没人知道在哪）
**命名规则**：意大利语/西班牙语/地方方言风格的恶搞双关名
```

### 2.3 主角家族设定：Belpaese 式模板
```markdown
**家族名**：[地名]家族（如：Belpaese → 改为：Spiceland / Pepperoni / 麻辣世家）
**特点**：
- 全员名字高度重复（Antonio, Antonia, Little Antonio, Uncle Antonio...）
- 每个人都有荒诞的职业背景（牧师、学生、退休屠夫、网红主播）
- 所有人都自称"猎手"，但没人见过真正的[反派]
**解锁方式**：地图上找到的棺材/冰箱/快递箱里不是反派，是新家族成员
```

## 三、角色设定（Character Design）

### 3.1 初始角色（免费解锁）
```markdown
### [名字] [姓氏]（如：Antonio Belpaese）
- **称号**：最后的希望 / 家族长子 / 被迫营业
- **背景**：
  "当[反派名]的诅咒降临时，[名字]正在[搞笑的日常场景，如：吃泡面/刷短视频/上厕所]。
   现在，他/她拿起[日常物品变成的武器]，踏上了复仇之路。"
- **性格**：严肃但无能 / 过度自信 / 根本不想来
- **初始武器**：[日常物品]（如：飞出的拖鞋、回旋的披萨、反弹的键盘）
- **特殊能力**：无 / 或一个搞笑的被动（如：+10%运气，因为"根本没打算活着回去"）
```

### 3.2 隐藏角色（地图解锁）
```markdown
### [名字] [姓氏] II / III / IV
- **解锁条件**：在[场景名]存活10分钟 / 打开[容器名]
- **背景**：
  "[名字]在[容器名]里睡了一觉，醒来发现世界变了。他/她决定加入战斗，
   尽管他/她完全不理解发生了什么。"
- **特点**：与初始角色外观几乎一样，只有颜色/配饰不同
- **元笑点**：玩家以为会解锁强力角色，结果只是换了个帽子
```

### 3.3 盟友角色（DLC/成就解锁）
```markdown
### [职业] [名字]（如：Sister Clerici / Father Dommario）
- **来源**：其他家族 / 神职人员 / 被诅咒的动物 / 敌方叛逃者
- **背景**：
  "[名字]原本是要来阻止[主角家族]的，但在看到[荒诞场景，如：会跳舞的骷髅]后，
   决定加入他们。'至少这里很有趣，'[名字]说。"
- **武器风格**：与职业强相关但荒诞化（牧师扔圣经 → 扔手机播放圣歌）
```

## 四、场景/关卡叙事（Stage Narrative）

### 4.1 场景叙事设计原则
1. **每个场景都是"寻找反派"的又一个错误地点**
2. **场景描述在加载界面显示，语气越来越绝望/荒诞**
3. **场景内的可互动物品（烤鸡/宝箱/棺材）承载叙事**

### 4.2 基础场景模板（6个标准关）

```markdown
### 场景1：[形容词]森林（如：Mad Forest → 改为：Hangry Forest 饥饿森林）
**加载界面文案**：
"[主角家族]闯入了[场景名]。传说[反派名]曾在这里[做了一件 mundane 的事，如：野餐/遛狗]。
森林里全是[怪物名]，但吸血鬼/反派？ nowhere to be found。"

**环境叙事元素**：
- 地上的烤鸡/食物（解释回血道具）
- 倒下的路牌指向错误方向
- 背景中若隐若现的[反派]剪影（走近发现是树/广告牌）

**隐藏叙事**：
- 存活10分钟后，背景音乐会加入[反派]的笑声（其实是乌鸦叫）
```

```markdown
### 场景2：[形容词]塔（如：Gallo Tower → 改为：Waffle Tower 华夫塔）
**加载界面文案**：
"也许[反派名]在塔顶？[主角家族]开始攀登。每一层都有更多怪物，
但塔顶只有一张纸条：'我去图书馆了。'"

**环境叙事元素**：
- 螺旋上升的塔身，背景在旋转（制造眩晕感）
- 每层有书架/冰箱/奇怪的家电
- 塔顶的"Boss区"空空如也，只有一个便签
```

```markdown
### 场景3：[形容词]图书馆（如：Inlaid Library → 改为：Infinite Library 无限图书馆）
**加载界面文案**：
"[主角家族]在图书馆寻找关于[反派名]的线索。他们找到了——
全是关于如何烹饪[怪物名]的食谱。至少晚餐有着落了。"

**环境叙事元素**：
- 无限延伸的书架（背景滚动）
- 漂浮的文字/弹幕（敌人攻击方式）
- 隐藏房间里的"禁书"（解锁新角色的线索）
```

```markdown
### 场景4：废弃[工厂/设施]（如：Dairy Plant → 改为：Bubble Tea Plant 奶茶工厂）
**加载界面文案**：
"这里曾是生产[日常食品]的地方。现在它生产[怪物名]。
[反派名]的邪恶真是...无处不在，又难以捉摸。"

**环境叙事元素**：
- 工业流水线变成怪物传送带
- 巨大的[食品]罐体（可破坏，掉落金币）
- 安全广播循环播放："请注意，[反派名]不在这里。"
```

```markdown
### 场景5：神圣/恐怖[建筑]（如：Cappella Magna → 改为：Capella Pasta 神圣意面堂）
**加载界面文案**：
"这是最后的希望。[主角家族]相信[反派名]一定在这里。
毕竟，还有什么地方比[神圣建筑]更适合躲藏呢？"

**环境叙事元素**：
- 哥特式建筑 + 现代霓虹灯（混搭风格）
- 彩色玻璃上描绘的不是圣徒，而是[家族成员]的搞笑事迹
- 祭坛上放着的不是反派，而是一面镜子（暗示玩家自己才是反派？）
```

```markdown
### 场景6：隐藏/奖励场景（如：Moongolow → 改为：Moonboba 月光珍珠）
**解锁条件**：在前五关各存活15分钟
**加载界面文案**：
"[主角家族]发现了一个隐藏维度。这里的一切都在发光，
包括那些[怪物名]。也许[反派名]就在这里？不，只有更多的金币。"

**环境叙事元素**：
- 迷幻色彩，低保真美学
- 天使/神灵形态的敌人（但还是很弱）
- 商人以全息投影形式出现
```

## 五、游戏内叙事系统（In-game Narrative System）

### 5.1 文本叙事节点
```markdown
**开场文本（游戏首次启动）**：
"20XX年，[地区名]。
[反派名]用[荒诞力量]诅咒了这片土地。
[主角家族]发誓要终结这一切。

...但他们首先需要找到[反派名]。
祝你好运。"

**死亡文本（Game Over）**：
"[主角家族]倒在了第[X]分钟。
[反派名]依然逍遥法外。
...如果[反派名]真的存在的话。"

**30分钟存活文本（通关）**：
"你存活了30分钟！
[死神/清算者]出现了，因为'时间到了'。
[主角家族]被带走了——但[反派名]呢？
也许下次吧。"
```

### 5.2 道具/武器叙事
```markdown
**武器命名规则**：
- 基础武器 = 日常物品 + 夸张形容词（如："愤怒的法棍"、"回旋的拖鞋"）
- 进化武器 = 神圣化/魔化日常物品（如："神圣披萨切片"、"无限奶茶杯"）

**武器描述文本示例**：
"[武器名]
传说[家族祖先]曾用它[做了一件 mundane 的伟大事迹]。
现在它归你了，别弄丢了。"

**被动道具描述**：
"[道具名]（如：Spinach 菠菜 → 改为：Instant Noodles 泡面）
'吃这个能变强。' ——[家族成员]，在战斗前夜"
```

### 5.3 敌人命名叙事
```markdown
**普通敌人**：
- [颜色/状态] + [ mundane 动物/物品]（如：Blue Bat → 改为：Angry Pigeon 愤怒鸽子）
- 命名越普通，数量越多，反差感越强

**精英敌人**：
- [形容词] + [职业] + [动物]（如：Giant Enemy Crab → 改为：Managerial Skeleton 经理骷髅）
- 暗示这些怪物生前也是打工人

**Boss敌人（每关隐藏）**：
- 不是[反派名]，而是[反派名]的[荒诞替身/使者/宠物]
- 击败后掉落："[反派名]的又一张便条：'我不在这里，别找了。'"
```

### 5.4 商人/NPC叙事
```markdown
**商人（每关开始/隐藏房间）**：
"我是商人。我有武器、有道具、有坏消息。
坏消息是：[反派名]不在这里。
好消息是：我的商品打九折。"

**交互文本**：
- 购买时："明智的选择。比寻找[反派名]靠谱多了。"
- 离开时不买："你也会像[反派名]一样躲着我吗？"
```

## 六、扩展剧情线（Expansion / DLC Lore）

### 6.1 DLC1：东方/异域篇（Legacy of the Moonspell 式）
```markdown
**标题**：[地区]的诅咒（如：Curse of the East）
**设定**：
[主角家族]听说[反派名]可能逃到了[东方地区]。
那里有一个守护[神秘山谷]的家族[XXX一族]，如今已经没落，
变成了[妖怪/当地怪物]的巢穴。

**新叙事元素**：
- 引入新的家族/盟友（当地守护者后裔）
- 怪物从哥特风转为东方神话风（但保持荒诞）
- 最终发现[反派名]的"东方分身"其实是个冒牌货
```

### 6.2 DLC2：学院/奇幻篇（Tides of the Foscari 式）
```markdown
**标题**：[学院名]的潮汐（如：Tides of the Academy）
**设定**：
[主角家族]来到[学院名]，一所教授[魔法/科技/烹饪]的学校。
学生们分属不同[学院/派系]，但地下封印着一个[古神/终极怪物]。

**新叙事元素**：
- 可操控角色变为学生/教师
- 剧情从"找反派"转为"阻止学生作死"
- 最终Boss是意外解封的[古神]，而[反派名]只是旁观者
```

### 6.3 DLC3：联动篇（Emergency Meeting / Contra 式）
```markdown
**标题**：意想不到的盟友（如：Unexpected Allies）
**设定**：
来自[其他IP/原创平行宇宙]的角色被卷入了[主角家族]的世界。
他们决定暂时放下恩怨，因为"这里的怪物比[原世界]还离谱"。

**叙事笑点**：
- 联动角色也在寻找自己的反派
- 两个世界的"找不到反派"困境产生共鸣
- 最终发现两个反派是同一个存在的不同化身（可选）
```

## 七、AI代码生成提示词（AI Implementation Prompts）

以下提示词可直接复制到 AI 代码生成工具（ChatGPT / Claude / Cursor 等）中，用于在生成游戏代码时注入剧情设定。

### Prompt 1：世界观注入（用于初始化项目）
```text
我正在开发一个受《Vampire Survivors》启发的网页端Roguelike生存游戏。

【世界观设定】
- 时间：20XX年，架空现代
- 地点：被诅咒的[Spiceland / 麻辣国]，一个哥特式乡村与现代网络文化混搭的地区
- 反派：Gourmando Feastó（高曼多·盛宴伯爵），一个被解雇的米其林主厨，
  他用"永恒饥饿诅咒"让大地陷入了荒诞的危机——到处都是食物，但所有人都在挨饿。
- 主角：The Spicy家族（辛辣家族），一个热爱美食但战斗能力可疑的家族，
  他们发誓要终结Gourmando的统治，尽管没人知道他躲在哪里。
- 核心荒诞设定：游戏表面上是"猎杀反派"，实际上反派可能不存在。
  每个关卡加载时的文案都要体现"又找错了地方"的幽默感。

【叙事风格要求】
- 哥特式荒诞幽默（Gothic Absurdist Comedy）
- 大量使用双关、反讽和meta-commentary（角色似乎知道自己在一个游戏里）
- 所有文本简短，适合网页端快速阅读
- 中文输出，但保留英文角色名和地名的恶搞感

请基于以上设定，生成游戏的【开场画面文案】和【第一个场景（Hungry Forest）的加载文案】。
```

### Prompt 2：角色系统注入（用于角色选择界面）
```text
基于以下角色设定，生成一个JavaScript角色数据对象（JSON格式），
包含初始3个可玩角色，每个角色需要：

【角色模板】
1. Antonio Spicy（安东尼奥·辛辣）
   - 称号：被迫营业的长子
   - 背景：诅咒降临时他正在吃泡面，现在他拿着飞出的拖鞋战斗
   - 初始武器：Bouncing Slipper（弹跳拖鞋）
   - 被动：+10%移动速度（因为"想快点结束回家"）

2. Imelda Spicy（伊梅尔达·辛辣）
   - 称号：家族里的虔诚者（但信仰的是美食之神）
   - 背景：她本想在教堂安静祈祷，但怪物打断了她的午餐
   - 初始武器：Spinning Ladle（旋转汤勺）
   - 被动：+15%经验获取（因为"好学"）

3. Gennaro Spicy（热纳罗·辛辣）
   - 称号：退休的屠夫，现在的"猎手"
   - 背景：他声称自己见过Gourmando，但所有人都知道他在撒谎
   - 初始武器：Boomerang Cleaver（回旋菜刀）
   - 被动：+20%力量（但-10%攻击速度，因为"年纪大了"）

【要求】
- JSON格式，包含：id, name, title, lore, weapon, passive, stats
- 每个角色的lore字段是一段50字以内的幽默背景
- 中文输出注释，但角色名和武器名保留英文
```

### Prompt 3：关卡叙事注入（用于场景系统）
```text
为网页端生存游戏生成5个关卡的配置数据，每个关卡需要以下叙事字段：

【关卡结构】
{
  id: string,
  name: string,           // 中文场景名
  nameEn: string,         // 英文恶搞名
  loadText: string,       // 加载界面文案（100字以内，体现"找错地方"的幽默感）
  ambientLore: string[],  // 环境叙事文本（3-5条，会随机显示在场景角落）
  bossHint: string,       // Boss战前提示（荒诞风格）
  clearText: string       // 存活30分钟后显示的文案
}

【5个关卡主题】
1. 森林关卡 → Hungry Forest（饥饿森林）
2. 塔关卡 → Waffle Tower（华夫塔）
3. 图书馆关卡 → Infinite Cookbook Library（无限食谱图书馆）
4. 工厂关卡 → Bubble Tea Plant（奶茶工厂）
5. 教堂关卡 → Capella Pasta（神圣意面堂）

【文案风格】
- 每段加载文案都要提到反派Gourmando Feastó，并暗示他可能在这里
- 每段结尾都要揭示"他不在这里"或给出更离谱的线索
- 环境叙事要包含对食物的荒诞描述（如：会跑的烤鸡、发光的珍珠奶茶）
```
